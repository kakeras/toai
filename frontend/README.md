# ToAI Frontend

## English

### Setup
1. Install dependencies:
```bash
npm install
```

### Development
1. Start the development server:
```bash
npm run dev:netlify
```
2. Open http://localhost:8888 in your browser

### Build
```bash
npm run build
```
